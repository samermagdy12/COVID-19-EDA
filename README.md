## ğŸš€ Setup and Run Instructions

### Prerequisites
*   Python 3.x

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/samermagdy12/covid_19_eda.git
    cd covid_19_eda
    ```
2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

### Running the Analysis
1.  Start a Jupyter Notebook server from the project root directory:
    ```bash
    jupyter notebook
    ```
2.  Open the `notebooks/covid_19_analysis.ipynb` file.
3.  Run the cells sequentially to reproduce the data cleaning, analysis, and visualizations.
